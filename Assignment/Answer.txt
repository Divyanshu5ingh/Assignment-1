Command to initialize git repository ==> git init

Command to clone remote repository ==> git clone <remote-URL>

Command to stage all changes ==> git add --all or git add .

Command to view state of working directory and staging area ==> git status

Command to create new branch ==> git branch <branch-name>

Command to change branch ==> git checkout <banch-name>

Command to clean staging area ==> git reset --hard

Command to get commits and changes from remote branch (It should not merge changes in current branch)==> git fetch <remote-URL> <branch-name>

Command to get commits and changes from remote branch (It should merge changes in current branch). ==> git pull <remote-URL> <branch-name>

Command to cherry pick commit which SHA => 092018283103810930  ==> git cherry-pick 092018283103810930

Command to push branch ==> git push origin <branch-name>

Command to delete remote branch ==> git push origin -d <banch-name>

Command to do iterative rebase ==> git rebase i <commit-base>
